The Inspector 🕵️‍♂️

The inspector is a built-in tool of the web browser.

To use the inspector:

  1. Right-click and select "Inspect".
  2. or use a keyboard shortcut:
    • Ctrl + Shift + I on Google Chrome
    • Ctrl + Shift + C on Mozilla Firefox

🕵️‍♂️ What does it do?

  • view the HTML code of the page...
  • and modify it!

Style = CSS

  1. Now, click the "Add Some Style" button.
  2. Observe the changes on the page.
  3. See also what happened in the inspector Styles tab

This is CSS code.

CSS stands for Cascading Style Sheets

Interactivity = JavaScript

  1. Click on Add a Dose of Interactivity .
  2. This adds the dynamic features of a to-do list.

JavaScript is the programming language responsible for the interactivity of a web page .

Find the CSS and JS Code

You can access files available on a website via their URL.

  • HTML Code index.html
  • CSS Stylesheet: style.css
  • JavaScript Code: script.js
