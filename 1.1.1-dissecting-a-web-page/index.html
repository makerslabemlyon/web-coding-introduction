How Does a Web Browser Work?

The browser allows displaying content accessible to users, from ... computer code .

Let's discover how this works.

Source Code

  1. Visit the website of la toile
  2. Right-click on the page.
  3. Select "View Page Source" (Google Chrome) or "Page Source" (Mozilla Firefox) .
  4. Observe the page that opens: it's HTML code .

This code is complex and difficult to read. So let's start with something simpler.

The Inspector 🕵️‍♂️

The inspector is a tool of the web browser.

It allows you to discover what's happening "under the hood" .

To use the inspector:

  1. Right-click and select "Inspect".
  2. Use a keyboard shortcut:
    • Ctrl + Shift + I on Google Chrome
    • Ctrl + Shift + C on Mozilla Firefox

The Inspector 🕵️‍♂️

The inspector allows you to view the HTML code of the page... and even modify it!

Content

As we've seen: the HTML code elements correspond to the different elements on the page.

The HTML code is responsible for the content of a web page

To display readable and aesthetic web interfaces, we need to then ... add some style .

Style

  1. Click the "Add Some Style" button.
  2. Observe the changes on the page.
  3. From the inspector, notice what appears in the Styles tab

This is CSS code.

CSS allows adding style to a web document

Add a Dose of Interactivity

  1. Click on Add a Dose of Interactivity .
  2. The page now allows you to interact with a to-do list.

This interactive functionality is made possible through the use of the programming language JavaScript .

JavaScript code allows adding dynamic functionality to a web page.

Find the CSS and JS Code

You can access files available on a website via their URL.

  • HTML Code index.html
  • CSS Stylesheet: style.css
  • JavaScript Code: script.js
