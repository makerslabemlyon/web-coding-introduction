What Makes Up a Web Page?

3 different languages to discover:

  • HTML: for the structure and content .
  • CSS: for the style - visual appearance & animation.
  • JavaScript: for interactivity - dynamic behavior.

Web Languages

HTML

HyperText Markup Language.

  • Manages the structure and content of the page.
  • Defines headings, paragraphs, images, links, etc.
  • Default structure can be modified via CSS.
  • Targets both human visitors and search engine robots.

HTML - Example


<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    <h1>Main Title</h1>
    <p>Paragraph of text.</p>
  </body>
</html>

CSS

Cascading Style Sheets.

  • Controls the visual appearance of HTML elements.
  • Modifies layout, colors, fonts, transparency, etc.
  • Adds visual animations.
  • Creates an attractive and consistent design.

CSS - Example


h1 {
  color: blue;
  font-size: 24px;
}

JavaScript

Manages the page's behavior.

  • Adds dynamic functionality: forms, animations, etc.
  • Allows retrieving real-time data.
  • Modifies page content interactively.

JavaScript - Example


document.getElementById('logo').style.maxWidth = '50px';

Try copying this line of code into the browser console , then press Enter.

Tools for Writing Code

To write code, you need a text editor.

  • Examples: Visual Studio Code, Sublime Text, Atom, etc.
  • Modern tools: syntax highlighting, auto-completion, etc.
  • Extensions to facilitate code writing.

Best Practices

  • Search for and use available tools.
  • Avoid reinventing the wheel.
  • Create on your own while making use of existing tools.
